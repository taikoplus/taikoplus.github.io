[日本語]
-必要なドライバープログラム-
7-ZIP / Visual C++ /  Rivatuner Statistics Server / VoiceMeeter

-インストールプロセス-
ゲームをスムーズにするために、4 つのプログラムでドライバーのインストールが必要です。（必ず管理者権限で実行してください。
1. Setup1.exeプログラムを実行して7-ZIPをインストールします。
2. Setup2.exeプログラムを実行して、Visual C++をインストールします。
3. Setup3.exeプログラムを実行して、Rivatuner Statistics Serverをインストールします。
(설치후 실행경로 : C:\Program Files (x86)\RivaTuner Statistics Server\RTSS.exe)
(タスクバーで小さなアイコンのRivatuner Statistics Serverをクリックし、フレームレートを120に調整します。)
4.Setup4.exeプログラムを実行してVoiceMeeterプログラムをインストールします。
インストール後、メニューをクリックしてRun on Windows Startup, System Trayをチェックします
すべてインストールが完了したら、Taiko no Tatsujin Nijiiro Plus Install.exeを実行してもいいです。

[English]
-required driver program-
7-ZIP / Visual C++ /  Rivatuner Statistics Server / VoiceMeeter

-Installation process-
Driver installation is required with 4 programs to facilitate the game (must run with administrator privileges).
1. Run the Setup1.exe program to install 7-ZIP.
2. Run the Setup2.exe program to install Visual C++.
3. Run the Setup3.exe program to install the Rivatuner Statistics Server.
(설치후 실행경로 : C:\Program Files (x86)\RivaTuner Statistics Server\RTSS.exe)
(Click the small icon Rivatuner Statistics Server in the task bar and adjust the frame rate to 120.)
4.Run the Setup4.exe program to install the VoiceMeeter program.
(After installation, click the manual to check Run on Windows Startup, System Tray.)
Once all have been installed, you are welcome to run Taiko no Tatsujin Nijiro Plus Install.exe.

[함국어]
-필요한 드라이버 프로그램-
7-ZIP / Visual C++ /  Rivatuner Statistics Server / VoiceMeeter

-설치과정-
게임을 원활하게 하기위해 4개의 프로그램으로 드라이버 설치가 필요합니다.(반드시 관리자 권한으로 실행하십시오.
1. Setup1.exe 프로그램을 실행하여 7-ZIP을 설치합니다.
2. Setup2.exe 프로그램을 실행하여 Visual C++를 설치합니다.
3. Setup3.exe 프로그램을 실행하여 Rivatuner Statistics Server를 설치합니다.
(설치후 실행경로 : C:\Program Files (x86)\RivaTuner Statistics Server\RTSS.exe)
(작업표시줄에서 작은 아이콘 Rivatuner Statistics Server를 클릭하고 프레임레이트를 120으로 조정합니다.)
4.Setup4.exe 프로그램을 실행하여 VoiceMeeter프로그램을 설치합니다.
(설치후 매뉴를 클릭하여 Run on Windows Startup , System Tray 를 체크합니다.)
모두 설치가 완료되면 Taiko no Tatsujin Nijiiro Plus Install.exe를 실행 하셔도 좋습니다.
